# 📚 课程作业管理系统

> 人工智能通识课 · 实践项目 | 第一类：智能体编程应用开发

## ✨ 项目简介

一个基于 Flask 的轻量级课程作业管理系统，支持学生提交作业、教师批改打分、管理员统一管理。

## 🎯 核心功能

### 学生角色
- 查看作业列表
- 在线提交作业（支持修改）
- 查看成绩和教师评语

### 教师角色
- 发布作业（标题、描述、截止时间）
- 查看学生提交列表
- 批改打分并填写评语

### 管理员角色
- 查看系统统计
- 添加/删除用户
- 导出所有数据为 JSON

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| Flask | Web 后端框架 |
| SQLite | 轻量级数据库 |
| HTML/CSS | 前端界面 |
| Claude Code | AI 辅助开发 |
